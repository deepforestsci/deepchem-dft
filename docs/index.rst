DeepChem DFT Project
==========================

**DeepChem DFT** is a pure python library for running DFT calculations.

Quick Start
-----------

Set up the conda environment and install the package:

.. code-block:: bash

    conda create -n dcdft python=3.14
    pip install -e .

About
-----

DeepChem DFT is part of the `DeepChem`_ family of scientific machine-learning tools.

.. _`DeepChem`: https://github.com/deepchem/deepchem
