Contributing
============

Contributions are welcome. Please open an issue or pull request on
`GitHub <https://github.com/deepforestsci/deepchem-dft>`_.

1. Fork the repository and create a feature branch.
2. Install development dependencies: ``pip install -r requirements/requirements-devl.txt``
3. Write tests for your changes and ensure the test suite passes.
4. Open a pull request against ``master``.
